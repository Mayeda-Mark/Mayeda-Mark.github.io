Back Door Bakery

This is a website/web application I made for my sister in-law's bakery. The bakery is now closed (unrelated to the website) which is why the site is only up on heroku. This application was built using Node.js, handlebars and pSQL. The application has a seperate section only accessible via logging in through session authentication. From this area, authorized users can update the blog and calendar events via tools that I built to access the pSQL database which stored the blog posts and calendar events.
