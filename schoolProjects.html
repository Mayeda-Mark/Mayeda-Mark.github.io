School Projects

Below, you will find varoius projects which I made for school and a brief explanation of what each project is:

Loop and conditional statements:

This applicaiton is built with vanilla front-end javaScript which allows a user to input a number to generate a multiplacation table of that size and input a number to view multiples of that number.


Objects:

This is a simple application where a user can enter a football team name and their win/loss record. The application then writes this information into an object and then reads the information back in a complete sentence.


JSON/Stringify:

This application is very similar to the objects application, except that it uses JSON and stringify() to generate the statement inputed by the user.


XMLHTTP:

This application makes a simple XMLHTTP request and displays some of the information found to the user.


Local storage:

This application is a demonstration of using local storage. The user inputs their name and refreshes the page. The page then greets the user by name. The user can also "pick a card" which will then be stored and read back to the user from local storage.


Domain manipulation:

Here, domain manipulation is used to generate a user's shopping list and a number of admitedly silly buttons.


CSS manipulation through javaScript:

This application uses javaScriptto change the page's CSS through user input.


Animate:

This application makes a simple animation from some text on the screen.


javaScript events:

This applicaiton uses javaScript events to make simple changes to the page's CSS.


Tags/Canvas:

This application demonstrates the use of video and audio tags in a web applicaiton and demonstrates the use of a canvas to make a smiley face.


Tansitions:

This applicaiton demonstrates the use of CSS transitions to manipulate html elements.


Transform:

This applicaiton demonstrates the use of CSS transforms to manipulate html elements.
